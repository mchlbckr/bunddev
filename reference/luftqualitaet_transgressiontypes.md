# List transgression types

List transgression types

## Usage

``` r
luftqualitaet_transgressiontypes(params = list(), safe = TRUE, refresh = FALSE)
```

## Arguments

- params:

  Query parameters.

- safe:

  Logical; apply throttling and caching.

- refresh:

  Logical; refresh cached responses.

## Value

A tibble with transgression type metadata.

## Details

Returns available transgression types. Official docs:
https://luftqualitaet.api.bund.dev.
