# List station settings

List station settings

## Usage

``` r
luftqualitaet_stationsettings(params = list(), safe = TRUE, refresh = FALSE)
```

## Arguments

- params:

  Query parameters.

- safe:

  Logical; apply throttling and caching.

- refresh:

  Logical; refresh cached responses.

## Value

A tibble with station settings metadata.

## Details

Returns station settings metadata. Official docs:
https://luftqualitaet.api.bund.dev.
