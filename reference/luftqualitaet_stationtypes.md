# List station types

List station types

## Usage

``` r
luftqualitaet_stationtypes(params = list(), safe = TRUE, refresh = FALSE)
```

## Arguments

- params:

  Query parameters.

- safe:

  Logical; apply throttling and caching.

- refresh:

  Logical; refresh cached responses.

## Value

A tibble with station types metadata.

## Details

Returns station types metadata. Official docs:
https://luftqualitaet.api.bund.dev.
