# Get data version

Get data version

## Usage

``` r
psm_stand(safe = TRUE, refresh = FALSE)
```

## Arguments

- safe:

  Logical; apply throttling and caching.

- refresh:

  Logical; refresh cached responses.

## Value

A tibble with version information.

## Details

Returns the release date/version of the plant protection product
database.

## Examples

``` r
if (FALSE) { # \dontrun{
psm_stand()
} # }
```
