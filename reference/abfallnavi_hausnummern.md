# List house numbers for a street

List house numbers for a street

## Usage

``` r
abfallnavi_hausnummern(strassen_id, safe = TRUE, refresh = FALSE)
```

## Arguments

- strassen_id:

  Street id.

- safe:

  Logical; apply throttling and caching.

- refresh:

  Logical; refresh cached responses.

## Value

A tibble with house numbers.
