# List Autobahn road ids

List Autobahn road ids

## Usage

``` r
autobahn_roads()
```

## Value

A tibble with available road ids.

## Details

Lists Autobahn road ids from the Autobahn App API (Autobahn GmbH).
Official docs: https://autobahn.api.bund.dev.

## See also

[`autobahn_roadworks()`](https://mchlbckr.github.io/bunddev/reference/autobahn_roadworks.md)
and
[`autobahn_warnings()`](https://mchlbckr.github.io/bunddev/reference/autobahn_warnings.md)
for road-specific data.

if (FALSE) { # \dontrun{
