# Get MOWAS archive mapping

Get MOWAS archive mapping

## Usage

``` r
nina_archive_mowas_mapping(identifier, safe = TRUE, refresh = FALSE)
```

## Arguments

- identifier:

  Warning identifier.

- safe:

  Logical; apply throttling and caching.

- refresh:

  Logical; refresh cached responses.

## Value

A tibble with archive mapping data.
