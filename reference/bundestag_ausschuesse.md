# List Bundestag committees

List Bundestag committees

## Usage

``` r
bundestag_ausschuesse(safe = TRUE, refresh = FALSE)
```

## Arguments

- safe:

  Logical; if `TRUE` (default), apply rate-limiting and cache GET
  responses to `tools::R_user_dir("bunddev", "cache")`.

- refresh:

  Logical; if `TRUE`, ignore cached responses and re-fetch from the API
  (default `FALSE`).

## Value

A tibble with one row per committee `<item>` from the index feed.

## Details

Returns the committee index from Bundestag XML feeds. API documentation:
<https://bundesapi.github.io/bundestag-api/>.
